# IT Support Cost Optimization Using Data Analysis

## Project Overview
This project analyzes IT support tickets using Python and Machine Learning to identify cost-saving opportunities and improve operational efficiency.

## Objectives
- Reduce IT support cost
- Improve customer satisfaction
- Reduce unnecessary escalations
- Optimize engineer allocation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Machine Learning Model
- Random Forest Regressor

## Features
- Data preprocessing
- Missing value handling
- Cost analysis
- Data visualization
- Feature importance analysis
- Cost prediction model

## Output
- support_tickets.csv
- cost_prediction_model.pkl
